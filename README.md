# Perflo user documentation (Mintlify)

This folder is a complete Mintlify docs site.

To preview locally:

1. npm i -g mint
2. cd into this folder
3. mint dev

To publish: push this folder to a GitHub repo and connect it in the Mintlify dashboard (mintlify.com). The site config is docs.json. Add your favicon.svg and logo files at the root.

Before going live, search and replace:
- perflo.ai links if your app or support domains differ
- support@perflo.ai if your support address differs
